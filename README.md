# Enterprise Business Analysis Portfolio

This repository contains an anonymized enterprise-level business analysis case study created for professional demonstration purposes.

The artifacts demonstrate practical business analysis experience in a regulated financial environment, including:
- Business and process requirements definition
- Functional and technical requirements specification
- Requirements elicitation and validation
- Business process and data modelling
- Impact and risk analysis
- Use case development
- Governance and delivery considerations
- System vision definition and modelling (Visio)

No proprietary systems, data, or confidential information are included.
